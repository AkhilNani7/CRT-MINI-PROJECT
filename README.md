# CRT-MINI-PROJECT
Title : Smart Parking Allocation Systems using Greedy Approach 
This is A Mini Project of Real Time Parking Allocation System for Vehicles using Greedy Algorithm
The Smart Parking Allocator is an intelligent, high-performance web application designed to solve modern urban parking congestion using advanced data structures and dynamic cloud pricing.
The project was developed in two distinct evolutionary phases:
Phase 1: Streamlit Baseline Prototype – A rapid Python-based terminal-like interface demonstrating simple business logic.
Phase 2: AI-Assisted Modern Cloud Architecture – A highly interactive Single Page Application (SPA) built with React 18, Vite, Tailwind CSS, custom priority queue data structures (Min-Heaps), and a serverless real-time cloud database (Google Cloud Firestore).
This report outlines the design principles, architectural transformations, algorithmic implementations, and database paradigms that drove this development cycle.
